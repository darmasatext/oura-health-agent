# 🏥 Oura Health Agent

---

## 🇬🇧 English

### Overview

The **Oura Health Agent** is an intelligent assistant that integrates with your Oura Ring to provide personalized health insights, proactive wellness monitoring, and conversational access to your biometric data through OpenClaw's agent framework.

### Key Features

- **📊 Real-time Health Monitoring**: Access your sleep, readiness, and activity scores on demand
- **🤖 Conversational Interface**: Ask questions about your health data in natural language
- **🔔 Proactive Insights**: Get notified about trends, anomalies, and personalized recommendations
- **📈 Historical Analysis**: Track patterns over days, weeks, or months
- **🎯 Goal Tracking**: Monitor progress toward your wellness objectives
- **🔒 Privacy-First**: Your health data stays secure and under your control

### Use Cases

- "How did I sleep last night?"
- "What's my readiness score today?"
- "Show me my activity trends this week"
- "Should I work out today?"
- "Remind me to wind down if my sleep score drops below 75"

### Quick Start

```bash
# Clone the repository
git clone https://github.com/diegoarmas/oura-health-agent
cd oura-health-agent

# Install dependencies
npm install

# Configure your Oura API token
cp .env.example .env
# Edit .env and add your OURA_API_TOKEN

# Run the agent
npm start
```
