🧪 Test Onboarding Modal

Herramienta para probar el modal de bienvenida del dashboard

📋 Instrucciones de Prueba:
  1. Click "Resetear" para simular primera visita
  2. Ir al dashboard principal: http://localhost:3000
  3. Verificar que modal aparezca después de 500ms
  4. Probar navegación (Anterior/Siguiente)
  5. Verificar que "Comenzar" cierre y guarde
  6. Recargar página → Modal NO debe aparecer
🛠️ Comandos para DevTools Console:
// Ver estado actual
localStorage.getItem('hasSeenOnboarding')

// Resetear
localStorage.removeItem('hasSeenOnboarding')

// Marcar como visto
localStorage.setItem('hasSeenOnboarding', 'true')
